# 🏠 Predictive Insight Engine (House Price Prediction)

## 📌 Project Overview
This project focuses on building and evaluating supervised learning models to predict house prices based on features like area, bedrooms, bathrooms, location score, and property age.

The project covers end-to-end machine learning workflow including data analysis, model building, evaluation, and optimization.

---

## 🎯 Objective
- Understand and implement supervised learning algorithms  
- Apply regression techniques for prediction  
- Analyze model performance using evaluation metrics  
- Study bias-variance trade-off, overfitting, and underfitting  

---

## 📊 Dataset Features
- House Area (sq.ft)  
- Number of Bedrooms  
- Number of Bathrooms  
- Location Score  
- Age of Property  
- House Price (Target Variable)  

---

## ⚙️ Technologies Used
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Scikit-learn  

---

## 🚀 Project Workflow

### 1. Data Preparation
- Handled missing values  
- Cleaned column names  
- Split dataset into training and testing sets  

### 2. Exploratory Data Analysis
- Pairplot visualization  
- Correlation heatmap  

### 3. Models Implemented
- Simple Linear Regression  
- Multiple Linear Regression  
- Polynomial Regression  

### 4. Model Evaluation
- Mean Squared Error (MSE)  
- Mean Absolute Error (MAE)  
- Root Mean Squared Error (RMSE)  
- R² Score  
- Adjusted R² Score  

### 5. Gradient Descent Optimization
- Batch Gradient Descent  
- Stochastic Gradient Descent (SGD)  
- Mini-Batch Gradient Descent  

---

## 📈 Results & Insights
- Multiple Linear Regression gave the best performance  
- Polynomial Regression showed overfitting at higher degrees  
- Simple Linear Regression showed underfitting  
- Feature scaling was essential for Gradient Descent  
- Mini-Batch Gradient Descent provided optimal performance  

---

## 🧠 Key Learnings
- Importance of feature selection  
- Impact of model complexity  
- Understanding bias-variance trade-off  
- Role of scaling in optimization algorithms
